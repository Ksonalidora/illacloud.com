## <h2 hidden>Co je REZERT API?</h2>

V ILLA Cloud funkce REST API umožňuje vývojářům interakci s externími systémy, Tyto prostředky jsou určeny na pokrytí výdajů na zaměstnance a správních výdajů agentury (hlavy 1 a 2) a provozních výdajů na pracovní program (hlava 3). S REST API v ILLA Cloudu mohou vývojáři bezproblémově integrovat své nízkoprogramové aplikace s jinými aplikacemi, databázemi nebo službami.

REST API v ILLA Cloud umožňuje vytvářet vlastní koncové body, které mohou zpracovávat HTTP požadavky, jako je GET, POST, PUT, DELETE. To umožňuje vývojářům provádět různé operace, včetně získávání dat, předávání údajů, aktualizace záznamů a mazání informací.

### Vytvořit REST API

Po přihlášení do vašeho účtu Illa existují dva způsoby, jak vytvořit zdroj v Illa.

-   Vytvořit ve zdrojích

Přihlaste se ke svému účtu Illa, vyberte `**Zdroje**` v horní části stránky a klikněte na tlačítko `**Vytvořit nové**`.

![external_resource](https://cdn.illacloud.com/official-website/img/docs/external_resource.png)

Vyberte **`REST API`** z databázového seznamu.

![action_list](https://cdn.illacloud.com/official-website/img/docs/action_list.png)

Připojte se k databázi s požadovanými parametry popsanými v části `Nastavení připojení` níže.

Klepněte na tlačítko `**Test Connection**` a zjistěte, zda se můžeme úspěšně připojit k databázi. Pokud ano, klikněte na **`Uložit zdroje`**, jinak dvakrát zkontrolujte hostname, port, uživatelské jméno a heslo je správné.

Po vytvoření zdroje se bude zobrazovat připravené RERERET API.

![rest_ex_ready](https://cdn.illacloud.com/official-website/img/official-site/components/rest_ex_ready.jpg)

-   Vytvořit ve staviteli

Přihlaste se ke svému účtu Illa, vytvořte projekt v Illa Builder na stránce `**App**` a přejděte na **`Seznam akcí`** v dolní části stránky. Klikněte na `**nové**`, pak vyberte **`REST API`** z databázového seznamu. Poté se připojte k databázi s požadovanými parametry popsanými v části `Nastavení připojení` níže.

Klepněte na tlačítko `**Test Connection**` a zjistěte, zda se můžeme úspěšně připojit k databázi. Pokud ano, klikněte na **`Uložit zdroje`**, jinak dvakrát zkontrolujte hostname, port, uživatelské jméno a heslo je správné.

### Konfigurovat REST API

| Vlastnosti      | Popis                                                                                                             |
| --------------- | ----------------------------------------------------------------------------------------------------------------- |
| Typ akce        | GET, POST, PUT, HEAD, PATCH, DELET, VOLBY                                                                         |
| Parametr URL    | poskytnout další informace nebo upravit chování požadavků API, pokud není specifikováno v nastavení připojení     |
| Záhlaví         | další metadata k žádosti o API nebo odpovědi pokud nejsou specifikována v nastavení připojení                     |
| Transformátor   | transformace dat do stylu, který se vám líbí pomocí JavaScriptu                                                   |

#### ZÍSKAT

načíst nebo načíst data ze serveru. Při podání žádosti o GET klient specifikuje adresu URL zdroje, ke kterému chce přistupovat, a server odpovídá požadovanými daty.

#### POST

vytvořit nový dokument na serveru. Umožňuje klientům odeslat data na server ke zpracování a uložení. Při odesílání žádosti o POST klient obsahuje údaje v těle žádosti.

#### PLOT

aktualizace nebo nahrazení existujícího zdroje. Obvykle se používá ke změně stavu nebo obsahu zdroje identifikovaného na základě konkrétní URL.

#### HLAVA

podobný metodě GET, ale s jedním klíčovým rozdílem: metoda HEAD načítá pouze hlavičky odezvy, bez útvaru pro odezvu.

#### PATCH

aktualizovat specifický zdroj. Obvykle se používá k částečným úpravám existujícího zdroje, aniž by se nahradil celý zdroj.

#### ODSTRANIT

požádá o odstranění konkrétního zdroje nebo údajů identifikovaných danou URL

### Případ použití

Ukážeme, jak automaticky vyplnit informace o formuláři z dat REST.

#### Krok 1 Přidat akci

Vytvořme tabulku v REST API se sloupci `id`, `name`, a `age`. Dokument id 1 je to, co chceme zobrazit ve složce formuláře.

![rest_get_all](https://cdn.illacloud.com/official-website/img/official-site/components/rest_get_all.jpg)

Pak můžeme vytvořit novou akci pro REST API ze seznamu akcí v aplikaci, kterou chceme vytvořit, výpis všech dat v kbelíku a jménem restapi1.

Vyberte `GET` pro `**Typ akce**`. Do URL parametrů vložte `id` pro klíč a `1` pro hodnotu**.**

Klikněte na **`Uložit`** a **`Spustit`** pro aktivaci této akce.

![rest_get_1](https://cdn.illacloud.com/official-website/img/official-site/components/rest_get_1.jpg)

#### Krok 2 Přidat komponenty

Přidejte komponentu **`form`** s výchozím textem "Form" a tlačítkem "Odeslat" pro plátno.

Přidejte komponentu **`input`**, komponentu **`number input`** a komponentu **`button`** z `**Vložit**` do formuláře.

Označte vstupní komponentu, která má být "Název", vstupní komponentu, která má být "Age", a komponentu tlačítka bude "Automatická plnění".

Nastavte zástupný symbol vstupní komponenty na "Zde prosím zadejte své jméno" a výchozí hodnotu vstupního čísla na 0, jak je uvedeno níže.

![rest_layout](https://cdn.illacloud.com/official-website/img/official-site/components/rest_layout.jpg)

#### Krok 3 Konfigurace komponenty

Pro složky "Automatická plnění" **\`tlačítko** můžeme nastavit hodnotu v komponentě **`input`**.

1.  Kliknutím na "Automatické vyplnění" **\`tlačítko** otevřete jeho stránku pro prohlížení. Pod **`Ovladač událostí`**, klikněte `**+ Nový**`.
2.  V obsluze události Editace vyberte **`Control component`** v akci, a vyberte komponentu **`input`**, kterou chcete aktualizovat jako cíl události.
3.  Vyberte akci **`setValue`** a nastavte hodnotu jako `{{restapi1.data[0]["name"]}}`

![rest_get_input](https://cdn.illacloud.com/official-website/img/official-site/components/rest_get_input.jpg)

Podobně, pro "Automatické vyplnění" **\`tlačítko** složky nastaví hodnotu v komponentě **`number input`**.

1.  Pod **`Ovladač událostí`**, klikněte `**+ Nový**`.
2.  V obsluze události Editace vyberte **`Control component`** v akci, a vyberte komponentu **`number input`**, kterou chcete aktualizovat jako cíl události.
3.  Vyberte akci **`setValue`** a nastavte hodnotu jako `{{restapi1.data[0]["age"]}}`

![rest_get_num](https://cdn.illacloud.com/official-website/img/official-site/components/rest_get_num.jpg)

Pro složku "Odeslat" **`button`** můžeme nakonfigurovat pro odeslání formuláře, pokud je formulář vyplněn

1.  Klepněte na komponentu **\`tlačítko** pro otevření její kontrolní stránky. Pod **`Ovladač událostí`**, klikněte `**+ Nový**`.
2.  V obsluze události Editace vyberte **`Control component`** v akci, a vyberte komponentu **`form`**, kterou chcete aktualizovat jako cíl události.
3.  Vyberte metodu **`submit`**

![rest_submit_config](https://cdn.illacloud.com/official-website/img/official-site/components/rest_submit_config.jpg)

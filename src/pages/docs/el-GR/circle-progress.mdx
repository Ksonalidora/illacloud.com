## <h2 hidden>Τι είναι ο Κύκλος προόδου?</h2>

Το στοιχείο προόδου κύκλου είναι ένα οπτικό στοιχείο που χρησιμοποιείται για την εμφάνιση της προόδου ή της κατάστασης ολοκλήρωσης σε κυκλική μορφή. Συχνά χρησιμοποιείται για να αντιπροσωπεύει την πρόοδο των καθηκόντων, των έργων ή οποιαδήποτε άλλη μετρήσιμη αξία.

### Ιδιότητες

| Ιδιότητες               | Περιγραφή                                                                                                                                    |
| ----------------------- | -------------------------------------------------------------------------------------------------------------------------------------------- |
| Τιμή                    | η ποσοστιαία τιμή αντιπροσωπεύει την πρόοδο                                                                                                  |
| Απόκρυψη ετικέτας τιμής | αν η ετικέτα τιμής είναι κρυμμένη                                                                                                            |
| Υπόδειξη                | Οι χρήστες μπορούν να εισάγουν εδώ την υπόδειξη συστατικού. Η υπόδειξη θα εμφανίζεται όταν εστιάζεται. Υποστηρίζεται μορφή Markdown          |
| Hidden                  | Δυναμική ρύθμιση της απόκρυψης του συστατικού. Μπορείτε να αλλάξετε την κρυφή κατάσταση μέσω μιας δυναμικής δυαδικής τιμής.                  |
| Ευθυγράμμιση            | Ορισμός της στοίχισης (στοίχιση αριστερά ή δεξιά) της ετικέτας                                                                               |
| Χρώμα Θέματος           | Επιτρέπει στους χρήστες να καθορίσουν το χρώμα φόντου και την αδιαφάνεια του κουμπιού                                                        |
| Styles                  | το πλάτος της διαδρομής του κύκλου αντιπροσωπεύει την πρόοδο                                                                                 |

### Μέθοδος

Μπορείτε να χρησιμοποιήσετε άλλα στοιχεία για τον έλεγχο της εφαρμογής. Υποστηρίζουμε τις ακόλουθες δύο μεθόδους:

-   **ορισμός**

Για να ορίσετε την τιμή εισόδου της περιοχής κειμένου, για παράδειγμα, {{"value1"}}

| Ιδιότητες | Περιγραφή   |
| --------- | ----------- |
| Τιμή      | Input value |

-   **καθαρή αξία**

Για να καθαρίσετε την τιμή του επιλεγμένου στοιχείου

### Δεδομένα

Το στοιχείο έχει κάποια κοινώς χρησιμοποιούμενα δεδομένα, τα οποία μπορούν να κληθούν μέσω του **`{{componentName.propertyName}}`** στην εφαρμογή.

| Όνομα ιδιότητας     | Περιγραφή                                                                  |
| ------------------- | -------------------------------------------------------------------------- |
| στοίχιση            | έλεγχο της θέσης του κύκλου προόδου                                        |
| χρώμα               | χρώμα του κύκλου                                                           |
| Εμφάνιση Ονόματος   | το όνομα αυτού του στοιχείου (δηλαδή το κουμπί1)                           |
| hidden              | κρυφή κατάσταση (αληθής ή ψευδές)                                          |
| πάχος               | πλάτος της διαδρομής του κύκλου                                            |
| tooltipText         | τιμή κειμένου υπόδειξης                                                    |
| Έγχρωμη             | χρώμα του μονοπατιού ή το γεμάτο τμήμα της προόδου του κύκλου              |
| τιμή                | ποσοστιαία τιμή της προόδου                                                |

Παράδειγμα: {{circleProgress1.value}}

### Περίπτωση χρήσης

Θα δείξουμε πώς να φτιάξουμε μια φόρμα με ένα στοιχείο προόδου κύκλο που δείχνει την πρόοδο ολοκλήρωσης της φόρμας.

#### Βήμα 1 Προσθήκη Στοιχείων

Πρόσθεσε ένα στοιχείο **`form`** με το προεπιλεγμένο κείμενο "Form" και ένα κουμπί "Submit" στον καμβά.

Πρόσθεσε ένα **`input`** component, ένα **`number input`** component, και ένα **`circle progress`** component από `**Insert**` στη φόρμα.

Ετικέτα το στοιχείο εισόδου να είναι "Όνομα", το στοιχείο εισόδου αριθμού να είναι "Ηλικία" και το στοιχείο προόδου μπάρα να είναι "Πρόοδος".

Ορίστε το σύμβολο κράτησης θέσης του στοιχείου εισόδου σε "Παρακαλώ εισάγετε το όνομά σας εδώ" και την προεπιλεγμένη τιμή εισόδου του αριθμού στο 0, όπως φαίνεται παρακάτω.

![cir_layout](https://cdn.illacloud.com/official-website/img/official-site/components/cir_layout.jpg)

#### Βήμα 2 Διαμόρφωση του εξαρτήματος

Για τα **`input`** και **`number input`** components, μπορούμε να τα ρυθμίσουμε ώστε να ορίσετε την τιμή σε **`circle progress`** component ώστε να αντικατοπτρίζει την πρόοδο της ολοκλήρωσης.

1.  Κάνε κλικ στο στοιχείο **`input`** και **`number input`** για να ανοίξεις την σελίδα ελέγχου της. Κάτω από **`Χειριστής Συμβάντων`**, κάντε κλικ στο `**+ Νέο**`.
2.  Στο χειριστή συμβάντων επεξεργασίας, επιλέξτε **`Control component`** εν ενεργεία, και επιλέξτε το στοιχείο **`circle progress`** που θέλετε να ενημερώσετε ως στόχο της εκδήλωσης.
3.  Επιλέξτε την **`setValue`** ενέργεια και ορίστε την τιμή ως

```jsx
{
  {
    input1.value && numberInput1.value
      ? "100"
      : input1.value || numberInput1.value
      ? "50"
      : "0";
  }
}
```

![cir_event_config](https://cdn.illacloud.com/official-website/img/official-site/components/cir_event_config.jpg)

Σημείωση: πρέπει να ρυθμίσετε τον χειριστή συμβάντος, όπως περιγράφεται παραπάνω, σε στοιχεία εισόδου BOTH και εισόδου αριθμού.

Για το στοιχείο "Υποβολή" **`κουμπί`** μπορούμε να το ρυθμίσουμε ώστε να υποβάλουμε τη φόρμα αν η φόρμα ολοκληρωθεί (δηλ. κύκλος προόδου 100%)

1.  Κάντε κλικ στο στοιχείο **`button`** για να ανοίξετε τη σελίδα ελέγχου. Κάτω από **`Χειριστής Συμβάντων`**, κάντε κλικ στο `**+ Νέο**`.
2.  Στο χειριστή συμβάντων επεξεργασίας, επιλέξτε **`Control component`** εν ενεργεία, και επιλέξτε το στοιχείο **`form`** που θέλετε να ενημερώσετε ως στόχο του γεγονότος.
3.  Επιλέξτε τη μέθοδο **`submit`** και βάλτε `{circleProgress1.value == 100}}}` στο **`Only run when`**

![cir_sub_config](https://cdn.illacloud.com/official-website/img/official-site/components/cir_sub_config.jpg)

Ομοίως, μπορούμε να ρυθμίσουμε το κουμπί για την εμφάνιση προειδοποίησης όταν ο χρήστης προσπαθεί να υποβάλει μια μη ολοκληρωμένη φόρμα.

1.  Κάντε κλικ στο στοιχείο **`button`** για να ανοίξετε τη σελίδα ελέγχου. Κάτω από **`Χειριστής Συμβάντων`**, κάντε κλικ στο `**+ Νέο**`.
2.  Στο χειριστή συμβάντων επεξεργασίας, επιλέξτε **`Show notification`** in action, **`title`** "Error!" και **`description`** "Παρακαλώ συμπληρώστε τη φόρμα πριν την υποβολή". Επιλέξτε το `Error` για το **`type`**, βάλτε το `{{2000}}` για το **`duration`** (2 δευτερόλεπτα), και βάλτε το `{{circleProgress1.value<100}}` για το **`Only run when`.**

![cir_err_config](https://cdn.illacloud.com/official-website/img/official-site/components/cir_err_config.jpg)

Μπορούμε επίσης να ρυθμίσουμε το στοιχείο `**Form**` για να εμφανίσουμε ένα μήνυμα επιτυχίας όταν υποβάλλουμε.

1.  Κάνε κλικ στο στοιχείο **`form`** για να ανοίξεις τη σελίδα ελέγχου. Κάτω από **`Χειριστής Συμβάντων`**, κάντε κλικ στο `**+ Νέο**`.
2.  Στο χειριστή συμβάντων επεξεργασίας, επιλέξτε **`Εμφάνιση ειδοποίησης`** σε δράση, **`title`** "Υποβλήθηκε με επιτυχία". Επιλέξτε το `Success` για **`type`**, βάλτε `{{2000}}` για **`duration`** (2 δευτερόλεπτα).

![cir_form_submit](https://cdn.illacloud.com/official-website/img/official-site/components/cir_form_submiy.jpg)

## <h2 hidden>Mitä REST API on?</h2>

ILLA Cloudissa REST API -toiminto antaa kehittäjille mahdollisuuden olla vuorovaikutuksessa ulkoisten järjestelmien kanssa, palvelut, ja tietolähteet hyödyntämällä edustusto State Transfer (REST) arkkitehtoninen tyyliä. REST API ILLA Cloudin avulla kehittäjät voivat integroida matalan koodin sovelluksensa saumattomasti muihin sovelluksiin, tietokantoihin tai palveluihin.

REST API ILLA Cloud mahdollistaa luoda mukautettuja päätepisteitä, jotka voivat käsitellä HTTP pyyntöjä, kuten GET, POST, PUT, ja DELETE. Tämän avulla kehittäjät voivat suorittaa erilaisia toimintoja, kuten hakea tietoja, toimittaa tietoja, päivittää tietueita ja poistaa tietoja.

### Luo REST API

On kaksi tapaa luoda resurssi Illa sen jälkeen, kun olet kirjautunut Illa tilillesi.

-   Luo resursseissa

Kirjaudu Illa-tilillesi, valitse `**Resurssit**` sivun yläreunassa ja napsauta `**Luo uusi **` -painiketta.

![external_resource](https://cdn.illacloud.com/official-website/img/docs/external_resource.png)

Valitse **`REST API`** tietokantaluettelosta.

![action_list](https://cdn.illacloud.com/official-website/img/docs/action_list.png)

Yhdistä tietokantaan vaaditut parametrit alla olevassa `Connection Settings`ssa.

Napsauta `**Testaa yhteyttä**` nähdäksesi, pystymmekö yhdistämään tietokantaan onnistuneesti. Jos kyllä, klikkaa **`Save Resources`**, muu, tarkista isäntänimi, portti, käyttäjätunnus ja salasana on oikea.

Kun resurssi on luotu, valmis REST API näyttää esitetyllä tavalla.

![rest_ex_ready](https://cdn.illacloud.com/official-website/img/official-site/components/rest_ex_ready.jpg)

-   Luo rakentajaan

Kirjaudu Illa tilillesi, luo projekti Illa Builderiin `**App**`-sivulle, ja siirry sivun alareunassa olevaan **`Action List`** -luetteloon. Valitse `**new**`, valitse **`REST API`** tietokantaluettelosta. Yhdistä tietokantaan sitten alla olevassa kohdassa `Connection Settings` kuvatut parametrit.

Napsauta `**Testaa yhteyttä**` nähdäksesi, pystymmekö yhdistämään tietokantaan onnistuneesti. Jos kyllä, klikkaa **`Save Resources`**, muu, tarkista isäntänimi, portti, käyttäjätunnus ja salasana on oikea.

### Aseta REST API

| Ominaisuudet       | Kuvaus                                                                                                            |
| ------------------ | ----------------------------------------------------------------------------------------------------------------- |
| Toiminnon Tyyppi   | KATSO, POST, PUT, HEAD, PATCH, POISTE, VAIHTOEHTO                                                                 |
| Url-Parametri      | antaa lisätietoja tai muuttaa API-pyyntöjen käyttäytymistä, jos sitä ei ole määritelty yhteysasetuksissa          |
| Otsikot            | muita metatietoja API-pyynnöstä tai vastauksesta, jos sitä ei ole määritetty yhteysasetuksissa                    |
| Muuntaja           | muuttamalla dataa tyyliin, jonka pidät JavaScriptin avulla                                                        |

#### LATAA

hae tai hae tietoja palvelimelta. Kun kyseessä on GET-pyyntö, asiakas määrittää sen resurssin URL-osoitteen, jota hän haluaa käyttää, ja palvelin vastaa pyydettyihin tietoihin.

#### POST

luo uusi resurssi palvelimelle. Sen avulla asiakkaat voivat lähettää tietoja palvelimelle käsitellään ja tallennetaan. POST-pyyntöä lähetettäessä asiakas sisällyttää siihen pyynnön sisältämät tiedot.

#### PUT

nykyisen resurssin päivittäminen tai korvaaminen. Sitä käytetään tyypillisesti tietyn URL-osoitteen avulla yksilöidyn resurssin tilan tai sisällön muuttamiseen.

#### PÄÄTTYÄ

samankaltainen kuin GET menetelmä, mutta yksi keskeinen ero: HEAD menetelmä hakee vain otsikot vasteen, ilman vastauselintä.

#### PATCH

päivittää tiettyä resurssia. Sitä käytetään tyypillisesti tekemään osittaisia muutoksia olemassa olevaan resurssiin korvaamatta koko resurssia.

#### POISTA

pyytää tietyn URL-osoitteen avulla yksilöidyn resurssin tai datan poistamista

### Käytä laatikkoa

Osoitamme miten auto täyttää lomakkeen tiedot REST API tiedot.

#### Vaihe 1 Lisää toiminto

Luodaan REST APIin taulukko, jossa on sarakkeet `id`, `name`, ja `age`. id 1 -asiakirja on se, mitä haluamme näyttää osana sitä.

![rest_get_all](https://cdn.illacloud.com/official-website/img/official-site/components/rest_get_all.jpg)

Sitten voimme luoda uuden toiminnon REST API -sovellusta varten toimintaluettelosta sovelluksessa, jonka haluamme rakentaa, luetellaan kaikki tiedot ämpärissä ja nimetään uudelleen 1.

Valitse `GET` `**Action Type**`. Laita URL-parametreihin `id` avaimelle ja `1` arvolle**.**

Napsauta **`Tallenna`** ja **`Suorita`** aktivoidaksesi tämän toiminnon.

![rest_get_1](https://cdn.illacloud.com/official-website/img/official-site/components/rest_get_1.jpg)

#### Vaihe 2 Lisää Komponentit

Lisää **`form`** komponentti oletustekstillä "Lomake" ja "Lähetä" -painikkeella kankaalle.

Lisää **`input`** komponentti, **`number input`** komponentti ja **`button`** komponentti `**Insert**` lomakkeeseen.

Merkitse syöttökomponentti "Nimi", numero tulo komponentti olla "Ikä" ja painikkeen komponentti on "Auto Fill".

Set the placeholder of the input component to "Please put your name here" and the default value of the number input to 0 as shown below.

![rest_layout](https://cdn.illacloud.com/official-website/img/official-site/components/rest_layout.jpg)

#### Vaihe 3 Määritä komponentti

Komponenttien "Auto Fill" **`button`** osalta voimme määrittää sen asettaaksemme arvon **`input`** komponenttiin.

1.  Klikkaa "Auto Fill" **`button`** -komponenttia avataksesi sen tarkasta sivun. Valitse **`Tapahtumien käsittelijä`**, klikkaa `**+ Uutta**`.
2.  Valitse Edit event handlerissa **`Ohjauskomponentti`** toiminnassa, ja valitse **`input`** komponentti, jonka haluat päivittää tapahtuman kohteeksi.
3.  Valitse **`setValue`** toiminto ja aseta arvo `{{restapi1.data[0]["name"]}}`

![rest_get_input](https://cdn.illacloud.com/official-website/img/official-site/components/rest_get_input.jpg)

Samoin "Auto Fill" **`button`** -komponenteille arvo asetetaan **`number input`** -komponenttiin.

1.  Valitse **`Tapahtumien käsittelijä`**, klikkaa `**+ Uutta**`.
2.  Valitse Edit event handlerissa **`Ohjauskomponentti`** toiminnassa, ja valitse **`number input`** komponentti, jonka haluat päivittää tapahtuman kohteeksi.
3.  Valitse **`setValue`** toiminto ja aseta arvo `{{restapi1.data[0]["age"]}}`

![rest_get_num](https://cdn.illacloud.com/official-website/img/official-site/components/rest_get_num.jpg)

Komponenttia "Lähetä" **`button`** varten voimme määrittää sen lähettääksemme lomakkeen, jos lomake on täytetty

1.  Napsauta **`button`** -komponenttia avataksesi sen tarkastussivun. Valitse **`Tapahtumien käsittelijä`**, klikkaa `**+ Uutta**`.
2.  Valitse Edit event handlerissa **`Ohjauskomponentti`** toiminnassa, ja valitse **`form`** komponentti jonka haluat päivittää tapahtuman kohteeksi.
3.  Valitse **`Lähetä`** -menetelmä

![rest_submit_config](https://cdn.illacloud.com/official-website/img/official-site/components/rest_submit_config.jpg)

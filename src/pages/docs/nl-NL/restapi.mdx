## <h2 hidden>Wat is REST API?</h2>

In ILLA Cloud zorgt de REST API functionaliteit ervoor dat ontwikkelaars kunnen communiceren met externe systemen, diensten en gegevensbronnen door gebruik te maken van de representatieve staatsoverdracht (REST) architectonische stijl. Met de REST API in ILLA Cloud kunnen ontwikkelaars hun low-codetoepassingen naadloos integreren met andere toepassingen, databases of diensten.

De REST API in ILLA Cloud maakt het mogelijk om aangepaste eindpunten te maken die HTTP-verzoeken kunnen verwerken, zoals GET, POST, PUT, en DELETE. Dit stelt ontwikkelaars in staat verschillende bewerkingen uit te voeren, waaronder het ophalen van gegevens, het verzenden van gegevens, het bijwerken van records en het verwijderen van informatie.

### REST API aanmaken

Er zijn twee manieren om een bron in Illa te maken nadat je bent ingelogd op je Illa account.

-   Maak in Bronnen

Log in op je Illa account, selecteer `**Resources**` bovenaan de pagina, en klik op de `**Maak Nieuw**` knop.

![external_resource](https://cdn.illacloud.com/official-website/img/docs/external_resource.png)

Selecteer **`REST API`** uit de database lijst.

![action_list](https://cdn.illacloud.com/official-website/img/docs/action_list.png)

Maak verbinding met de database met de vereiste parameters beschreven in `Connectie Instellingen` hieronder.

Klik op `**Test Connection**` om te zien of we met succes verbinding kunnen maken met de database. Zo ja, klik **`Resources`**, anders controleer je de hostnaam, port, gebruikersnaam en wachtwoord correct.

Na het aanmaken van een bron wordt de REST API weergegeven zoals getoond.

![rest_ex_ready](https://cdn.illacloud.com/official-website/img/official-site/components/rest_ex_ready.jpg)

-   Maak in Bouwer

Log in op je Illa account, maak een project aan in Illa Builder in de `**App**` pagina en navigeer naar de **`Action List`** onderaan de pagina. Klik op `**nieuw**`, en selecteer dan **`REST API`** uit de database lijst. Maak vervolgens verbinding met de database met de vereiste parameters beschreven in `Connectie Instellingen` hieronder.

Klik op `**Test Connection**` om te zien of we met succes verbinding kunnen maken met de database. Zo ja, klik **`Resources`**, anders controleer je de hostnaam, port, gebruikersnaam en wachtwoord correct.

### Gebruik geval

We zullen laten zien hoe we de formulierinformatie uit REST API-gegevens automatisch kunnen invullen.

#### Stap 1 Voeg een actie toe

Laat ons een tabel aanmaken in REST API met de kolommen `id`, `name`, en `age`. Het document id 1 is wat we willen laten zien in de vorm component.

![rest_get_all](https://cdn.illacloud.com/official-website/img/official-site/components/rest_get_all.jpg)

Daarna kunnen we een nieuwe actie voor REST API maken vanuit de actielijst in de app die we willen bouwen. Overzicht van alle gegevens in de bucket met de naam restapi1.

Selecteer `GET` voor `**Action Type**`. Gebruik in URL Parameters, plaats `id` voor sleutel en `1` voor waarde**.**

Klik **`Opslaan`** en **`Run`** om deze actie te activeren.

![rest_get_1](https://cdn.illacloud.com/official-website/img/official-site/components/rest_get_1.jpg)

#### Stap 2 Componenten toevoegen

Voeg een **`form`** component toe met de standaard tekst "Form" en een "Verzenden" knop voor canvas.

Voeg een **`input`** component toe, een **`number input`** component en een **`button`** component van `**Insert**` in het formulier.

Label het invoercomponent "Naam", het nummerinvoercomponent dat "Ouder" moet zijn en het knopcomponent "Auto vullen".

Stel de plaatshouder van het invoercomponent in op "Plaats uw naam hier" en de standaardwaarde van de invoergegevens op 0 zoals hieronder weergegeven.

![rest_layout](https://cdn.illacloud.com/official-website/img/official-site/components/rest_layout.jpg)

#### Stap 3 Configureer het component

Voor de "Auto Fill" **`button`** componenten, kunnen we het configureren om de waarde in **`input`** component in te stellen.

1.  Klik op het onderdeel "Automatisch vullen" **`button`** om de inspectiepagina te openen. Onder **`Event handler`**, klik `**+ Nieuw**`.
2.  In de Event Handler Bewerken Selecteer **`Control component`** in actie, en selecteer de **`input`** component die u wilt bijwerken als doel van het event.
3.  Kies de **`setValue`** actie en stel de waarde in als `{{restapi1.data[0]["name"]}}`

![rest_get_input](https://cdn.illacloud.com/official-website/img/official-site/components/rest_get_input.jpg)

Ook voor "Auto Vill" **`button`** componenten om waarde in te stellen in **`number input`** component.

1.  Onder **`Event handler`**, klik `**+ Nieuw**`.
2.  In de Event Handler Bewerken Selecteer **`Control component`** in actie, en selecteer het onderdeel **`number input`** dat u wilt bijwerken als doel van het event.
3.  Kies de **`setValue`** actie en stel de waarde in als `{{restapi1.data[0]["age"]}}`

![rest_get_num](https://cdn.illacloud.com/official-website/img/official-site/components/rest_get_num.jpg)

Voor het "Verzenden" **`button`** component, kunnen we configureren om het formulier in te dienen als het formulier is ingevuld

1.  Klik op de **`button`** component om de inspectie pagina te openen. Onder **`Event handler`**, klik `**+ Nieuw**`.
2.  In de Event Handler Bewerken Selecteer **`Control component`** in actie, en selecteer het onderdeel **`form`** dat u wilt bijwerken als doel van het event.
3.  Kies de **`submit`** methode

![rest_submit_config](https://cdn.illacloud.com/official-website/img/official-site/components/rest_submit_config.jpg)

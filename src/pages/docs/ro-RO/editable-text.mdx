## <h2 hidden>Ce este editarea textului?</h2>

Textul editat permite utilizatorilor să creeze și să manipuleze textul în aplicațiile lor. Cu această componentă, utilizatorii pot introduce și edita cu ușurință textul direct pe interfață, făcându-l ideal pentru a crea conținut dinamic și interactiv.

### Proprietăți

| Proprietăți                       | Descriere                                                                                                                                                                                |
| --------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Valoare implicită                 | Valoarea inițială a componentei. Puteți schimba dinamic valoarea inițială tastând JavaScript în {{}}.                                                                                    |
| Substituent                       | Valoarea va fi afișată când câmpul de intrare este gol.                                                                                                                                  |
| Etichetă                          | Numele câmpului afişat utilizatorului                                                                                                                                                    |
| Legendă                           | O legendă folosită pentru a descrie câmpul în detaliu                                                                                                                                    |
| Etichetă ascunsă                  | Setează dacă se afișează eticheta                                                                                                                                                        |
| Poziție                           | Setați poziția etichetei în raport cu componenta                                                                                                                                         |
| Aliniere                          | Setați alinierea (aliniere la stânga sau la dreapta) a etichetei                                                                                                                         |
| Width                             | Când eticheta este pe partea stângă a componentei, setaţi raportul lăţimii etichetei.                                                                                                    |
| Gestionar de evenimente           | Declanșează interogări, controlează componente sau apelează alte API-uri ca răspuns la evenimentele componentei.                                                                         |
| Dezactivat                        | Controlează starea de dezactivare a componentei. Componenta nu poate fi modificată sau focalizată atunci când este dezactivată.                                                          |
| Doar citire                       | Controlează starea dacă componenta este doar în citire. O componentă doar-în-citire poate fi selectată și focalizată, dar nu poate fi modificată.                                        |
| Afişare buton ştergere            | controlează dacă în câmpul de intrare este sau nu afișat un buton de ștergere                                                                                                            |
| Arată numărul de caractere        | controlează dacă este sau nu afișat numărul de caractere al intrărilor                                                                                                                   |
| Prefix text                       | un text scurt care apare în partea stângă a câmpului de intrare, adesea folosit pentru a oferi un context suplimentar sau instrucțiuni utilizatorului                                    |
| Text în completare                | un text scurt care pare în dreapta câmpului de intrare, utilizat adesea pentru a furniza informații suplimentare sau feedback utilizatorului                                             |
| Tooltip                           | Utilizatorii pot introduce aici pontul componentei. Sfatul de instrumente va fi afișat când este concentrat. Formatul markdown este acceptat.                                            |
| Câmp obligatoriu                  | Valid numai când comutatorul este pornit.                                                                                                                                                |
| Model                             | specifica un model pe care valoarea de intrare trebuie să îl corespundă                                                                                                                  |
| Lungime maximă                    | setează numărul maxim de caractere care pot fi introduse în câmpul de introducere. (Automatic Translation)                                                                               |
| Lungime minimă                    | specifica numărul minim de caractere pe care utilizatorul trebuie să îl introducă în câmpul de introducere pentru ca intrarea să fie considerată validă.                                 |
| Regulă personalizată              | Crează-ți logica validării aici. Regulile ar trebui să fie stabilite în JavaScript și acoperite de {{}}.                                                                                 |
| Ascunde mesajul de validare       | Puteți ascunde eroarea prin comutarea stării ascunse atunci când valoarea de intrare este incorectă. Poți schimba dinamic starea ascunsă de JavaScript.                                  |
| Hidden                            | Controlează dinamic dacă componenta este ascunsă. Puteți schimba starea ascunsă prin valoarea booleană dinamică.                                                                         |
| Culoare temă                      | Permite utilizatorilor să specifice culoarea de fundal şi opacitatea butonului                                                                                                           |

### Metodă

Puteţi utiliza alte componente pentru a controla componenta. Sprijinim următoarele cinci metode:

-   **setValue**

Pentru a seta valoarea de intrare a zonei de text, de exemplu, {{"value1"}}

| Proprietăți  | Descriere   |
| ------------ | ----------- |
| Valoare      | Input value |

-   **clearValue**

Pentru a șterge valoarea componentei selectate

-   **focus**

Când se apelează metoda focalizată, câmpul de introducere a zonei de text va fi evidențiat și gata pentru ca utilizatorul să înceapă să tasteze fără a fi nevoie ca utilizatorul să facă clic pe câmpul de introducere a numărului

-   **validate**

Pentru a verifica dacă informațiile de intrare sunt legale

-   **clearValidate**

Pentru a șterge mesajul de validare

### Gestionar de evenimente

| Eveniment   | Descriere                                                                                  |
| ----------- | ------------------------------------------------------------------------------------------ |
| Schimbă     | Când un utilizator schimbă valoarea de intrare                                             |
| Focalizare  | Când un utilizator mută cursorul mouse-ului pe componenta de intrare                       |
| Blur        | Când un utilizator a terminat de pus valoare şi a renunţat să se concentreze pe componentă |

### Date

Componenta are niște date utilizate în mod obișnuit, care pot fi apelate prin **`{{componentName.propertyName}}`** în aplicație.

| Nume proprietate             | Descriere                                                                                                      |
| ---------------------------- | -------------------------------------------------------------------------------------------------------------- |
| valoare                      | valoare de intrare utilizator                                                                                  |
| culori                       | Culoarea de fundal a butonului                                                                                 |
| regulă personalizată         | Regulă sau logică de validare definită de utilizator care poate fi aplicată componentei de intrare a numărului |
| dezactivat                   | o valoare Booleană indică starea persoanelor cu handicap                                                       |
| AfișeazăNume                 | numele acestei componente (de ex. buton1)                                                                      |
| evenimente                   | generat sau declanșat când apar anumite acțiuni sau evenimente                                                 |
| hidden                       | stare ascunsă (adevărată sau falsă)                                                                            |
| etichetă                     | valoare etichetă                                                                                               |
| Aliniere                     | alinierea etichetei de intrare (stânga sau dreapta)                                                            |
| Pozitie                      | eticheta de intrare (stanga sau dreapta)                                                                       |
| Lățime etichetă              | numărul întreg reprezentând lăţimea etichetei.                                                                 |
| lungime maximă               | valoarea lungimii maxime                                                                                       |
| Lungime minimă               | valoarea lungimii minime                                                                                       |
| substituent                  | valoare substituent                                                                                            |
| prefixText                   | valoarea prefixului textului                                                                                   |
| Sufix                        | valoarea sufixului textului                                                                                    |
| citire                       | o valoare Booleană indică starea de pregătire a intrărilor.                                                    |
| necesar                      | a Valoarea booleană indică starea necesară a intrărilor.                                                       |
| regex                        | aplică validarea bazată pe model la textul de intrare în zona de text                                          |
| număr de showCharacter       | o valoare Booleană indică dacă numărul de caractere de intrare este afișat                                     |
| tooltipText                  | valoarea textului cu înclinare unelte                                                                          |

Exemplu: {{editable_text1.value}}

### Utilizare majusculă

Apoi, vom demonstra cum să cartografiem datele din sursa de date la textul editabil cu un buton.

#### Pasul 1 Adăugare acţiune

Haideţi să creăm un tabel în REST API cu coloanele `id`, `name`, şi `age`. Documentul id 1 este ceea ce dorim să arătăm în forma componentei.

![rest_get_all](https://cdn.illacloud.com/official-website/img/official-site/components/rest_get_all.jpg)

Apoi putem crea o nouă acțiune pentru REST API din lista de acțiuni din aplicația pe care dorim să o construim, listarea tuturor datelor în găleată și denumită restapi1.

Selectați `GET` pentru `**Tipul de acțiune**`. În parametrii URL, puneți `id` pentru cheie și `1` pentru valoare**.**

Faceţi clic pe **`Salvează`** şi **`Run`** pentru a activa această acţiune.

![rest_get_1](https://cdn.illacloud.com/official-website/img/official-site/components/rest_get_1.jpg)

#### Pasul 2 Adaugă Componente

Apoi, putem adăuga o componentă **`editable text`** şi o componentă **`button`** la pânză.

Etichetăm textul editabil pentru a fi 'Nume' și am etichetat butonul ca "Setează valoarea" ca fiind afișat mai jos

![edit_layout](https://cdn.illacloud.com/official-website/img/official-site/components/edit_layout.jpg)

#### Pasul 3 Configurarea componentei

Pentru componenta **`button`**, îl putem configura pentru a seta valoarea în componenta **`editable text`** la datele documentului pe care îl citim din restapi.

1.  În gestionarul de editare a evenimentului, selectați **`Componenta Control`** în acțiune, și selectați **componenta editabilă a textului\`** pe care doriți să o actualizați ca țintă a evenimentului.
2.  Alegeți acțiunea **`setValue`** și setați valoarea ca `{{restapi1.data[0]["name"]}}`

![edit_event_config](https://cdn.illacloud.com/official-website/img/official-site/components/edit_event.jpg)

#### Etapa 4 Test

Acum că atunci când faceți clic pe butonul "Setare valoare", valoarea ar trebui să se schimbe în text.

![edit_test](https://cdn.illacloud.com/official-website/img/official-site/components/edit_test.jpg)

## <Heading hidden>Custom API authentication</Heading>

ILLA 默认支持多种身份验证标准（查看更多）。
此外，ILLA有一个Custom Auth身份验证模式，允许您组合多个步骤来支持任何API身份验证。
这些认证方法可用于REST和GraphQL资源。
